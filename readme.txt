== Description ==

The Post View Count plugin is a simple WordPress plugin designed specifically to display post view couns in post details page and admin column.

  
 **How does it work?**

== Install the post-view-count.zip file and activate it.

== Check the output from the single/details post page to see if it appears after the post content.
== Also, the post view count show in the admin column in All posts araea
== The column is sortable
== It shows the post view count with the shortcode also, the shortcode is [post_view_count id="22"] , you have to change the post id.

== Key Feature's  ==
 
 * It displays post view count in post single page .
 * It displays post view count in Admin Coulumn
 * The Admin Column is sortable
 * It shows the post view count for a specific post by  shortcode  ( with the post ID) and the shortcode is :[post_view_count id="22"]
